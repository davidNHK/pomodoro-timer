# Productivity app

- Status: accepted

## Context and Problem Statement

I want app that contain all tools i need to boost my productivity
app may contain:

- TODO list
- Pomodoro timer
- Mind map

It should support mobile device as well.

## Decision Drivers

- Stack familiarity
- Mobile support
- Hosting cost

## Considered Options

- Monolith web app
- Monolith mobile app
- Micro app with app shell

## Decision Outcome

- Monolith web app with possible micro app shell
